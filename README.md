# Crypto Price Monitor v1.0
Monitorea precios de criptomonedas en tiempo real con alertas personalizables.

## Features
- Monitorea BTC, ETH, SOL, y cualquier par de Binance
- Alertas configurables por % de cambio
- Sin API key requerida (datos públicos)
- Exportable a CSV

## Uso
```bash
python3 crypto-monitor.py --coins BTC,ETH,SOL --alert 2% --interval 60
```
