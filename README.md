Monitor cryptocurrency prices in real-time with customizable alerts — no exchange API key required.

## ✨ Features

- 🔥 **Live tracking** of BTC, ETH, SOL, and any Binance trading pair
- ⚡ **Configurable alerts** by % price change (1%, 2%, 5%, custom)
- 📊 **CSV export** of all monitored prices
- 🆓 **No API key needed** — uses Binance public data
- 🐧 **Cross-platform** — works on Linux, macOS, Windows

## 🖥️ Usage

```bash
python3 crypto-price-monitor.py
```

Press `Ctrl+C` to stop monitoring.
